---
name: KnowledgeArchivist
description: 在学习结束时，生成详细的学习总结和知识库归档。
---

# 触发条件
当用户明确表示“我学会了”、“任务完成”或请求总结时触发。

# 执行逻辑
1. **回顾上下文**：回顾所有的教学步骤、用户的代码、遇到的错误及修正过程。
2. **生成报告**：在 `book/` 目录下创建一个新的 markdown 文件（例如 `book/summary_fastapi_01.md`）。

# 输出内容规范
文件内容必须包含：
1. **知识点胶囊**：核心概念的表格。
2. **实战代码**：从 `test/` 目录中提取最终成功的代码片段。
3. **踩坑记录**：记录过程中真实的报错和解决方法。
4. **下一步建议**。

# 结束动作
生成文件后，告诉用户：“总结已归档到 book/ 文件夹，你可以随时查看。”
